M5Sequencer Pro v1.0
High-Performance 16-Step Groovebox for M5Stack Core2 & Tough

M5Sequencer Pro is a sophisticated 16-step sequencer and synthesizer designed for the M5Stack ecosystem. It combines PCM sample playback with a custom-built subtractive synthesis engine, providing a complete mobile music production environment with a focus on low-latency performance and intuitive touch control.

🚀 Core Features
4-Track Drum Machine: High-fidelity PCM (.WAV) playback for Kick, Snare, Hi-hat, and Clap via SD card.

Dual-Channel Synthesizer: Independent Bass and Melody tracks utilizing a Square Wave oscillator.

Advanced Legato Engine: Intelligent note-linking logic that eliminates transients between consecutive identical notes for smooth, professional synth lines.

2-Octave Interactive Keyboard: Optimized piano-style layout for real-time performance and melody composition.

Slot-Based File System: 6 dedicated memory slots for saving and loading patterns directly to SD/MMC (.tsq binary format).

Real-Time Performance Mixer: Per-track volume control and global BPM adjustment (40–250 BPM).

Power Management: Integrated battery level monitoring for mobile reliability.

🛠 Installation & Setup
SD Card: Format your SD card to FAT32.

Samples: Place kick.wav, snare.wav, hihat.wav, and clap.wav in the root directory.

Compilation: Flash the firmware using Arduino IDE or PlatformIO. Ensure the M5Unified library is installed.

Audio: Use the built-in speaker or the 3.5mm jack (on supported modules) for output.
